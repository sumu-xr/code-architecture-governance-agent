# 代码架构治理 Agent

## 项目简介
本项目是一款自动化代码架构治理Agent，针对存量/增量代码进行静态+语义分析，自动识别架构异味、技术债，并生成可执行的重构方案，最终输出治理报告与PR提案，帮助团队高效完成代码质量管控。

## 核心流程
1.  **代码拉取**：支持Git仓库拉取/本地代码目录导入
2.  **静态+语义扫描**：语法结构分析、依赖关系解析、架构模式识别
3.  **技术债分级**：按严重程度分级标记架构异味、代码坏味道
4.  **重构方案生成**：针对不同问题提供可落地的重构策略与代码示例
5.  **PR自动提交**：生成重构分支并自动创建Pull Request
6.  **自动化测试验证**：执行单元测试/集成测试，验证重构后的代码稳定性
7.  **治理报告输出**：生成可视化治理报告，包含问题清单、修复率、架构健康度评分

## 技术栈
- 语言：Python
- 核心依赖：AST语法树分析、GitPython、pytest
- 输出格式：Markdown报告、JSON数据、PR提案

## 使用说明
1.  配置仓库地址与认证信息
2.  执行架构治理命令：`python main.py --repo <你的仓库地址>`
3.  查看生成的重构报告与PR链接
